¿Cómo aparece el login de Tesla en la app?

Elige la estructura que prefieras para el flujo de autenticación

A

Página de login separada

Si no hay sesión activa, Streamlit muestra solo la pantalla de login. Una vez autenticado, redirige al dashboard completo.

Sin sesión → Login
🚗
Tesla Tracker
B

Sidebar con estado de sesión

El dashboard siempre se ve, pero el sidebar muestra "Conectar con Tesla". Si no hay sesión, los datos son los mocks. Si hay sesión, muestra datos reales.

C

Modal de login al cargar

El dashboard carga normalmente, pero aparece un modal/banner prominente al inicio pidiendo conectar la cuenta. Se puede omitir para ver datos de ejemplo.
